使用Angular

    简单的命令
  1. ng new my-app ：创建一个my-app的项目(新的工作空间)
  2. ng serve --open：自动打开浏览器，并访问 http://localhost:4200/
  3. app-root:是根组件。组件是Angular应用的基本构造块。
      一个组件包括三部分
    • app.comment.html
    • app.comment.css
    • app.comment.ts
  4. ng generate component heroes：创建一个heroes的组件
      如何让一个组件显示？
    • 把heroes添加到壳组件AppContent。
    • app-heroes 就是 HeroesComponent的元素选择器.
    • 即把《app-heroes》添加到AppContent中
      一个组件需要创建新的类Hero 就要创建一个hero.ts
    • 使用的时候在HeroesComment里导入 即import { Hero } from '../hero';
  5. [(ngModel)]：Angular 的双向数据绑定语法
  6. 在使用双向绑定前，必须在一个可选模块 FormsModule下执行
  7. AppModule:
    • Angular 需要知道如何把应用程序的各个部分组合到一起，以及该应用需要哪些其它文件和库。 这些信息被称为元数据（metadata）。
    • 有些元数据位于 @Component 装饰器中，你会把它加到组件类上。 另一些关键性的元数据位于 @NgModule 装饰器中。
    • 最重要的 @NgModule 装饰器位于顶级类 AppModule 上。
    • Angular CLI 在创建项目的时候就在 src/app/app.module.ts 中生成了一个 AppModule 类。 这里也就是你要添加 FormsModule 的地方。
    • 在app.module.ts中导入FormModel,然后把 FormsModule 添加到 @NgModule 元数据的 imports 数组中，这里是该应用所需外部模块的列表。
  8. 每个组件都必须声明在（且只能声明在）一个 NgModule 中。在app.module.ts导入各个组件，在@NgModule.declarations 数组中声明。
  9. 路由：1import路由的包2.创建路由对象3.调用对象 实现跳转 this.router.navigateByUrl("work") 4.配置路由的路径 app-routing.module.js
