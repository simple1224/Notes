布局容器

  1. bootstrap为页面和栅格系统创建了一个.container的容器

    2. .container 用于固定宽度和并且支持响应式布局的类(会居中显示)

    .container-fluid 用于100%宽度 占据全部窗口(满屏显示)

栅格系统

    Bootstrap 提供了一套响应式、移动设备优先的流式栅格系统，随着屏幕或视口（viewport）尺寸的增加,
    系统会自动分为最多12列。它包含了易于使用的预定义类，还有强大的mixin 用于生成更具语义的布局。
  1. 简言之就是利用行和列的组合来实现页面排版
  2. 行row只有在container和container-fluid中生效 更方便于添加合适的排列（aligment）和内补（padding）
  3. 通过行在水平方向设置一组列 也就是说内容一般写在列（column）中 列是行的直接子元素
  4. 可以通过预定义的类来快速创建栅格布局 .row .col-xs-4
  5. ??? 通过给列设置padding属性来设置列与列直接的间隔(gutter) 通过为 .row 元素设置负值 margin 从而抵消掉为 .container 元素设置的 padding，也就间接为“行（row）”所包含的“列（column）”抵消掉了padding.
  6. 负值的 margin就是下面的示例为什么是向外突出的原因。在栅格列中的内容排成一行。
  7. 栅格系统中的列是通过指定1到12的值来表示其跨越的范围。例如，三个等宽的列可以使用三个 .col-xs-4 来创建。
  8. 如果一“行（row）”中包含了的“列（column）”大于 12，多余的“列（column）”所在的元素将被作为一个整体另起一行排列。
  9. ???栅格类适用于与屏幕宽度大于或等于分界点大小的设备 ， 并且针对小屏幕设备覆盖栅格类。 因此，在元素上应用任何 .col-md-* 栅格类 用于与屏幕宽度大于或等于分界点大小的设备 ， 并且针对小屏幕设备覆盖栅格类。 因此，在元素上应用任何 .col-lg-* 不存在， 也影响大屏幕设备。

媒体查询 (media query)

    首先了解什么是媒体查询：
  1. @media 标签可以说是响应式网页的开发基础。其主要由媒体类型（Media Type）和 媒体特性（Media Query）两部分组成。
  2. media type:设定后面规则生效的展示类型，包括all（全部）,screen（屏幕）,print（页面打印或打邱预览模式）等等
  3. Media Query:设置后面生效的条件
  4. CSS语法: 
                @media Media Type and|not|only (Media Query) {
                CSS-Code;
            }
  5. 争对不同的媒体应用不同的stylesheet: <link rel="stylesheet" media="mediatype and|not|only (Media Query)" href="mystylesheet.css">

    6. 注:在html里面添加代码 可以使用pre和code标签。如果遇到标签时应该使用转义字符 “lt gt"表示一对尖括号 "quot"表示双引号

  6. 媒体查询类型有:all、screen、print等
  7. 媒体特性：color,width,max-width,min-width,scan等很多
  8. 媒体查询一般用于创建分界线的阈值
    1. 手机:对于小于768px的手机屏幕 没有特别设置 (原因是bootstrap是优先移动设备的)
    2. 小屏幕平板:@media (min-width: @screen-sm-min) { ... }
    3. 中屏幕:@media (min-width: @screen-md-min) { ... }
    4. 大屏幕:@media (min-width: @screen-lg-min) { ... }
    5. 或者在两个尺寸之间的:@media (min-width: @screen-sm-min) and (max-width: @screen-sm-max) { ... }

流式布局容器

  • 将最外面的布局元素.container改为.container-fluid

    • 移动设备和桌面屏幕

    • 如果不希望在小屏幕上列显示的时候是堆叠的 那就在定义时使用针对小、中屏幕设备所定义的类。即.col-xs-* .col-md-*。 针对超小屏幕是:.col-xs-*,小屏幕:.col-sm-*,中屏幕是:col-md-*,大屏幕的类是:.col-lg-*
    • 示例1:

      1个.col-xs-6表示在小屏幕中这个栅格占其余剩下的部分

      .col-xs-12 .col-md-8
      .col-xs-6 .col-md-4
    • 示例2:

      3个.col-xs-6表示在小屏幕中每个栅格占33.33%

      .col-xs-6 .col-md-4
      .col-xs-6 .col-md-4
      .col-xs-6 .col-md-4
    • 示例3:

      2个.col-xs-6表示在小屏幕中每个栅格占50%

      .col-xs-6 .col-md-4
      .col-xs-6 .col-md-4
